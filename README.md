# GIF Analyzer

## Requirements

```
customtkinter>=5.2.1
Pillow>=10.1.0
```

## Installation

1. Clone the repository
2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

### GUI Application
```bash
python gif_analyzer.py
```

### Command Line
```bash
python gif_parser.py path/to/file.gif
```

Flags:
- `-o, --output`: Save result to specified file
- `-h, --help`: Show help message
